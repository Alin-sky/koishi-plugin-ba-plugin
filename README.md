# koishi-plugin-ba-plugin
## koishi插件，bluearchive.碧蓝档案工具箱，正在不断开发新功能


## 功能
- 抽卡模拟器
- 角色评分
- 国际服千里眼
- 总力战模拟
- 礼物查询


## 声明
 - 数据来自于[bawiki](https://ba.gamekee.com/entry)
